어떤 마스코트 방향이 마음에 드세요?

지금 픽셀아트 톤은 유지하면서, "과하지 않은 귀여움"을 어디서 가져올지 골라주세요. 색상은 지금 배너의 GitHub 다크 팔레트(초록/보라/빨강)를 그대로 씁니다.

A. 카메라봇

RTSP/영상 추론을 그대로 캐릭터화. 빨간 REC 안테나가 포인트. 지금 하시는 일과 가장 직결됨.

B. 모니터 뒤 고양이

모니터 위로 빼꼼 보이는 보라색 고양이 + 스트림 파형. 범용적으로 가장 "귀여운 개발자 프로필" 감성.

C. GPU 칩 캐릭터

멀티-GPU 추론 플랫폼을 칩 얼굴로 표현. 우측 상단 초록 활동 표시등이 포인트.

D. 카메라봇 → 터미널 (지금 배너 계승)

지금 배너의 "카메라 → 터미널" 구도는 유지하되 챙비화. 스토리는 그대로, 톤만 귀엽게.

여러 개 마음에 들면 다중 선택도 가능해요 (예: A 캐릭터 + D 구도). 클릭해서 골라주시고, 터미널에도 코멘트 남겨주세요.
